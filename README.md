# 👁️ BLINDAI — Perceive with Intelligence

A futuristic assistive ecosystem designed to empower the visually impaired using the power of **Gemini AI** and **Computer Vision**.

---

## 🌟 Overview
**BlindAI** transforms the visual world into an audible and tactile experience. Built with a **Voice-First** philosophy, it allows users to navigate their surroundings, read text, and identify objects through intuitive AI-driven interactions.

---

## ✨ Key Features

- **🎙️ Voice Command Center** — Natural language interaction for all app functions.
- **📸 Intelligent Scene Description** — Detailed, context-aware analysis of your environment.
- **📄 Instant OCR Engine** — Real-time text-to-speech for signs, documents, and labels.
- **🎨 Color Specialist** — Precise color detection to assist in daily activities.
- **📳 Haptic UI** — Tactile feedback patterns for seamless navigation.
- **☁️ Secure Cloud Sync** — Activity history and settings backed by Firebase.

## 📐 Architecture
The project follows **Domain-Driven Clean Architecture** to ensure scalability and maintainability:

- **🔵 Domain Layer**: Use cases and business entities.
- **🟢 Data Layer**: Repositories, Data Sources (Firebase, AI APIs).
- **🔴 Presentation Layer**: BLoC logic and high-contrast UI components.

## 🚀 Getting Started

1. **Clone the project**
   ```bash
   git clone https://github.com/yourusername/blind_ai.git
   ```

2. **Install Dependencies**
   ```bash
   flutter pub get
   ```

3. **Firebase Setup**
   - Add your `google-services.json` (Android) or `GoogleService-Info.plist` (iOS) to the root folders.

4. **Run the Engine**
   ```bash
   flutter run
   ```
