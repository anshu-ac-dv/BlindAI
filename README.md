## 🌌 The Vision

> "To bridge the gap between visual impairment and independent living through the lens of Artificial Intelligence."

**BlindAI** isn't just an app; it's a sensory extension. Designed with a **Voice-First** philosophy and a **Deep Dark** aesthetic, it transforms complex visual environments into meaningful audio narratives.

## 🏗️ Technical Architecture

Built on the foundations of **Clean Architecture**, BlindAI is divided into three scalable layers:

- **🔵 Domain Layer**: Pure business logic & Use Cases.
- **🟢 Data Layer**: Firebase, ML Kit, and Repository implementations.
- **🔴 Presentation Layer**: BLoC State Management & Material 3 UI.

## 🚀 Getting Started

1. **Clone & Install**
   ```bash
   git clone https://github.com/yourusername/blind_ai.git
   cd blind_ai
   flutter pub get
   ```

2. **Configure Firebase**
   - Place your `google-services.json` (Android) or `GoogleService-Info.plist` (iOS) in the appropriate directories.

3. **Launch**
   ```bash
   flutter run
   ```
